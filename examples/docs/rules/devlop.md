# 前端开发文档

### 技术栈

* 打包工具: webpack5
* 前端框架: vue2, vue-router，vuex
* 样式预编译: Less
* UI 库: 
  + PC端: aex-ui (自建), 
  + 移动端: vant
* 网络请求: axios
* 国际化: vue-i18n
* 基础缓存: vuex-persistedstate
* 日期处理: dayjs
* 常用工具: lodash
* 图表库: echarts
* 轮播图: swiper
* 截图工具: html2canvas
* 模拟数据: mockjs
* CDN服务: 腾讯云 【加速域名：https://cdn.aex.news】

### 静态资源服务

webpack 生产环境打包时，静态资源路径要替换地址为: https://aex1.yxds.net.cn

```js
